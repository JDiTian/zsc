# 简介
训练集，测试集，验证集均在dataset文件夹下
由于电脑配置受限，GPU存储不够，无法利用GPU运行，且云端服务器成本较高，本次实验使用500张图片作为训练集利用CPU进行训练，50个epoch
训练结果在runs文件夹下的train中的exp33文件夹下
